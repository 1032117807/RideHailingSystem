# TripVerse RAG 知识库文档

这批文档用于后续接入 RAG 检索系统，内容以 TripVerse 平台内部规则为准，适合作为：

- 平台规则问答
- 购票资格问答
- 实名认证与账号状态问答
- 下单、支付、退款的规则说明

## 建议的切割方式

- 按二级标题切割：每个 `##` 作为一个主 chunk
- 长段再按三级标题切割：每个 `###` 作为一个子 chunk
- 控制单个 chunk 在 `300-600` 中文字左右
- 保留标题、适用对象、关键词字段，便于召回和重排

## 使用建议

- 如果是“站内规则”和“资格”问答，优先召回 `01 / 02 / 03`
- 如果是“是否能买、怎么买、为什么买不了”，优先召回 `02 / 03 / 04`
- 如果是“能不能退、退款状态、退款条件”，优先召回 `05`
- 如果是离线评测和 prompt 调试，可直接使用 `06`
